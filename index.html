How to Use Command Center

What is this?

Command Center keeps your family coordinated. Track all your athletes' practices, games, and tournaments in one place. Share the schedule code with family members so everyone knows where to be and when. Import entire seasons in minutes instead of hours!

Quick Start Guide

1

Create or Join a Schedule

Either create a new schedule to get a 6-character code, or join an existing one using a code someone shared with you.

2

Import Your Season Schedule

Go to Add Event → Import Schedule to add your entire season at once using Photo Import, Recurring Schedule, or manual entry.

3

View & Navigate Your Schedule

The This Week tab shows this week's events. Use arrows to navigate weeks and filter by athlete. All Games tab shows everything in table format.

Import Methods

📸 Photo Import (Recommended)

Snap a photo → Copy our prompt → Paste in ChatGPT or Gemini → Import your schedule. Done!

🔁 Recurring Schedule

Coach says "practice is Tuesdays & Thursdays at 5:30"? Just tell us the details and we'll build your whole season.

✏️ Add One by One

Enter events manually with an easy grid. Add rows, fill out details, and import when ready. Best for a few quick additions.

Pro Tips

  • • Start with Photo Import - fastest way to import a full season from photos
  • • Use Recurring Schedule - generate entire seasons when coach only gives you "Tuesdays at 5:30"
  • • Edit or delete events using pencil/trash icons in This Week or All Games tabs
  • • Share your 6-character code with coaches, grandparents, and siblings
  • • Use the Pack List and Weekly Notes to stay organized each week
  • • Filter by athlete when you have multiple kids to manage
  • • Navigate weeks with arrow buttons to see future games and practices

App Setup Instructions

Install as App

Install this on your device for instant access. No app store needed - it works like a native app right from your browser!

1. Open Safari and visit this schedule

2. Tap the Share icon at the bottom center

3. Select "Add to Home Screen"

4. Tap "Add" in the top right corner

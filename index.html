How to Use Command Center

What is this?

Command Center keeps your family coordinated. Track all your athlete's practices, games, and tournaments in one place. Share the schedule code with family members so everyone knows where to be and when.

Getting Started

1

Create or Join a Schedule

Either create a new schedule to get a code, or join an existing one using a code someone shared with you.

2

Add Events in Control Tab

Go to the Control tab and add practices, games, tournaments, or other events. Tell us who's going, when, where, and who's taking them.

3

View Your Schedule

The Schedule tab shows this week's events organized by day. Use the arrow buttons to see next week or go back.

4

See Everything in Season Tab

The Season tab shows all events for the whole season in a table format so you can see the big picture.

Pro Tips

  • • You can edit or delete any event by clicking the pencil or trash icons
  • • Share your code with coaches, grandparents, and siblings to keep everyone in sync
  • • Use the Pack List to remember everything you need for the week
  • • Add weekly notes to communicate important reminders to everyone
  • • Filter by athlete to focus on one kid's schedule
